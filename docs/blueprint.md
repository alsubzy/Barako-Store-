# **App Name**: MarketFlow Pro

## Core Features:

- Mock User Authentication: Secure (mock) login/logout system with role-based access for Admin and Staff users, allowing controlled access to features.
- Interactive Dashboard: A dynamic overview presenting key performance indicators like total sales, orders, and critical low stock alerts, complemented by mock revenue charts for quick insights.
- Comprehensive Product Management: Tools for creating, viewing, updating, and deleting products, including essential details such as name, category, price, stock quantity, unit type, and an optional product image.
- Smart Inventory Control: Real-time stock level tracking with visual highlights for low stock items and a dedicated modal for quick and efficient inventory adjustments.
- Efficient Point of Sale (POS) System: A streamlined interface for cashiers to search products, add them to a virtual cart, adjust quantities, calculate final totals, simulate the checkout process, and generate a printable receipt UI.
- Order & Customer Hub: A centralized module providing a detailed view of all past orders and basic customer information, including mock capabilities for order status updates.
- AI Product Description Assistant: A generative AI tool that assists staff by suggesting engaging and detailed product descriptions based on input attributes like category and name, enhancing product listings.

## Style Guidelines:

- The visual scheme embraces a light palette, focusing on clarity and professional freshness, reminiscent of a modern market. The primary color is a deep teal-green (#4A7770), chosen for its versatility and calming professional presence. The background color is a very light mint (#E7F7F4), providing a clean, airy foundation. A muted forest green (#629969) serves as the accent, offering an analogous contrast that remains understated yet effective for key interactive elements.
- The interface will utilize 'Inter', a contemporary sans-serif typeface, for all text elements. Its objective, neutral aesthetic ensures high readability and reinforces the clean, minimal, and professional SaaS dashboard design, suitable for both headlines and body content.
- Drawing from ShadCN UI's ecosystem, simple and modern line icons will be consistently used throughout the application. Icons will maintain a clean, minimalistic style, aligning with the overall professional dashboard aesthetic.
- The application will feature a modern SaaS dashboard layout, including a persistent sidebar navigation for core modules. Content areas will prominently utilize cards and tables for structured data display. All forms, consistently, will open within centered modal dialogs. The entire interface will be fully responsive across various screen sizes to ensure accessibility and usability on any device.
- Subtle and functional animations will be incorporated to enhance user experience, including smooth transitions for loading states to provide visual feedback and toast notifications for system alerts and confirmations.
